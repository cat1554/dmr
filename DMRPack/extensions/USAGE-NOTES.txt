To use the extensions, simply copy or move the .zip for 
the extension you want to use and put in the 
Minecraft resourcepacks folder like a normal pack. 

Make sure the extensions are above the full pack 
in the in-game settings resource list.

Extensions included in this pack:
  - Zero Hour: Moves 00:00 to dawn (/time set 0)
